Стрик без пропусков, Сначала собрать мусор, потом соревнование на точность бросков в корзину, Бросок виртуального кубика каждый день — выпадает случайное задание (собрать 10 окурков, отказаться от пластика и т.д.), Игра вроде тетриса или match-3, где собираешь/сортируешь падающий мусор под таймер.

## Тип моего проекта:
**«Мусорный Квест»** — игровой веб-сайт для экологических челленджей и соревнований по сбору мусора.

Основные механики:
- соревнования и челленджи (таймер, сортировка, upcycling)
- ежедневный сбор мусора со стриком
- бросок кубика с случайными заданиями
- браузерная игра (типа тетриса или match-3 с мусором)
- рейтинг и достижения

## Библиотеки и технологии:
- **HTML, CSS, JavaScript** — фронтенд
- **Python** (Flask или FastAPI) — бэкенд
- **SQLite** / PostgreSQL — база данных

## Референсы:
- Gamification с JavaScript
- Браузерный Tetris / Match-3 на Canvas
- Daily streak система на Flask + JS
